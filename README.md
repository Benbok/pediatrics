# VaxTrack RF - Педиатрический Календарь Прививок

Desktop приложение для отслеживания вакцинации детей в соответствии с законодательством РФ. Включает в себя локальную базу данных и интеллектуального помощника на базе Google Gemini.

## 📁 Структура приложения

### 🔹 Корень проекта
- `index.html` — Основной HTML-шаблон приложения. Здесь настроены Tailwind CSS, шрифты и карты импорта (import maps).
- `vite.config.ts` — Конфигурация сборщика Vite: настройки сервера, алиасы путей (`@`) и проброс переменных окружения.
- `package.json` — Список зависимостей, скрипты запуска приложения и настройки Electron.
- `App.tsx` — Центральный компонент React: управляет навигацией и состоянием активного профиля ребенка.
- `.env.local` — Файл для хранения ваших API-ключей (например, Gemini). Не передается в Git для безопасности.
- `.env.example` — Образец структуры файла окружения.

### 🔹 `electron/` — Системная часть (Main Process)
- `main.cjs` — Точка входа Electron. Управляет созданием окна, жизненным циклом приложения и политикой безопасности (CSP).
- `preload.cjs` — Безопасный мост между системными функциями и браузером. Экспортирует `window.electronAPI`.
- `database.cjs` — Весь бэкенд приложения: создание SQLite таблиц, SQL-запросы для сохранения профилей детей и записей о прививках.

### 🔹 `src/` — Интерфейс (Renderer Process)
- `modules/` — Функциональные блоки приложения (Feature-based architecture):
  - `patients/` — Регистрация и выбор профилей детей.
  - `calendar/` — Основной календарь прививок с расчетом сроков.
  - `dashboard/` — Сводка по предстоящим и пропущенным вакцинам.
- `components/` — Общие элементы интерфейса:
  - `layout/` — Оболочка приложения (Sidebar, Header, Main view).
  - `ui/` — Переиспользуемые кнопки, карточки и модальные окна.
- `services/` — Внешняя логика:
  - `geminiService.ts` — Интеграция с ИИ: медицинские справки и советы по догоняющей вакцинации.
- `types.ts` — TypeScript модели данных (ChildProfile, VaccineRecord и т.д.).
- `index.tsx` — Точка монтирования React-приложения.
- `index.css` — Глобальные стили и кастомные CSS-анимации.

## 🚀 Как запустить

### 1. Установка
```bash
npm install
```

### 2. Настройка ИИ (Опционально)
Создайте `.env.local` и добавьте:
```
VITE_GEMINI_API_KEY=ваш_ключ
```

### 3. Запуск в режиме разработки
```bash
npm run electron:dev
```

## 🛠 Технологический стек
- **Frontend**: React 18 + TypeScript + Tailwind CSS
- **Shell**: Electron 39
- **DB**: SQLite3
- **AI**: Google Gemini Pro 1.5 (через @google/genai)

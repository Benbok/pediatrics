# PediAssist - AI Agent Quick Reference Card

## ARCHITECTURE (СТРОГО!)
```
Component → Service → IPC Handler → Database
(View)      (Logic)   (Auth+Valid)  (Data)
```

## ПРАВИЛА (NEVER VIOLATE)

### 1. Validation: ДВОЙНАЯ
- Frontend Service: `Schema.parse(data)` ← UX
- Backend IPC: `Schema.parse(data)` ← Security

### 2. Authentication: ОБЯЗАТЕЛЬНА
```javascript
ipcMain.handle('db:*', ensureAuthenticated(async (_, data) => {
  // your code
}));
```

### 3. Transactions: Для связанных операций
```typescript
await prisma.$transaction(async (tx) => { /* ... */ });
```

### 4. Logging: ТОЛЬКО logger
```javascript
const { logger, logAudit } = require('./logger.cjs');
logger.error('[Module] Error:', error);
logAudit('OPERATION_NAME', { details });
```

### 5. NO Business Logic in Components!
```typescript
// ❌ Component с логикой → ✅ patientService.method()
```

## СТРУКТУРА ФАЙЛОВ
- Page/Module: `src/modules/<name>/`
- Service: `src/services/<name>.service.ts`
- Validator: `src/validators/<name>.validator.ts`
- Types: `src/types.ts`
- Constants: `src/constants.ts`

## ЗАПРЕЩЕНО
- `console.log` → Используй `logger.*`
- Magic numbers → Используй `constants.ts`
- IPC без auth → Оберни в `ensureAuthenticated`
- Функции без типов → Добавь TypeScript types

## РАЗМЕРЫ
- File < 500 строк
- Component < 300 строк
- Function < 50 строк

## ПРИМЕРЫ

### Service
```typescript
export const myService = {
  async create(data: Input): Promise<Output> {
    const validated = Schema.parse(data);
    return await window.electronAPI.createSomething(validated);
  }
};
```

### IPC Handler
```javascript
ipcMain.handle('db:create', ensureAuthenticated(async (_, data) => {
  const validated = Schema.parse(data);
  const result = await prisma.model.create({ data: validated });
  logAudit('CREATED', { id: result.id });
  return result;
}));
```

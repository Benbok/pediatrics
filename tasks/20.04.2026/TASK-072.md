# TASK-072 — Улучшение резервного копирования: бэкап при закрытии + проверка изменений

**Статус:** ✅ Завершена  
**Приоритет:** Средний  
**Дата создания:** 2026-04-20  

## Контекст

Текущая логика (`shouldBackupToday`) создаёт максимум один бэкап в день — при первом запуске.  
Проблема: если пользователь активно работал весь день и закрывает приложение, последний бэкап может быть утренним.

## Цель

1. Добавить `createBackupIfChanged(dbPath)` в `electron/backup.cjs` — создаёт бэкап только если `.db`-файл новее последнего бэкапа.
2. Вызывать функцию через `app.on('before-quit')` в `electron/main.cjs` — бэкап при каждом закрытии, но только при реальных изменениях.

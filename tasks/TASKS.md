## 📌 Правила работы с этим файлом

### Статусы задач
- 🔄 `IN_PROGRESS` — задача в работе прямо сейчас
- 📋 `PLANNED` — запланирована, ещё не начата
- ⏸️ `PAUSED` — приостановлена (блокер или переключение)
- ✅ `DONE` — полностью выполнена
- ❌ `CANCELLED` — отменена

### Нумерация ID
Формат: `TASK-NNN` (трёхзначный, с ведущими нулями: `TASK-001`, `TASK-012`)

### Как AI-агент работает с этим файлом

**Старт задачи:**
1. Прочитать этот файл
2. Определить следующий ID (max существующий + 1)
3. Создать папку `tasks/DD.MM.YYYY/` (сегодняшняя дата)
4. Создать `tasks/DD.MM.YYYY/TASK-NNN.md` с полным описанием
5. Добавить строку в раздел "Активные задачи" со статусом 🔄

**Завершение задачи:**
1. Переместить строку из "Активных" в "Выполненные"
2. Сменить статус на ✅
3. Обновить `README.md` затронутого модуля
4. Зафиксировать финальный статус в `TASK-NNN.md`

## 📋 Описание задачи

Под ответом ИИ-поиска (`KnowledgeQueryWidget`) добавить блок **«Препараты»**, где найденные medication-источники отображаются как кликабельные чипы-кнопки с переходом на `/medications/:id`.

### Контекст

- В `result.sources` уже есть объекты `{type: 'disease'|'medication', name, id}`.
- Компонент `SourceLink` уже умеет навигировать на `/medications/:id` и `/diseases/:id`.
- Текущий блок «Источники из базы знаний» смешивает болезни и препараты в одном списке.
- `useNavigate` уже импортирован и используется в `SourceLink`.

### Ожидаемый результат

- Новый блок «Упомянутые препараты:» с Pill-иконкой виден только когда в `sources` есть хотя бы один `type === 'medication'`.
- Клик на чип-препарат → `navigate('/medications/:id')`.
- Блок «Источники из базы знаний» остаётся, но показывает только болезни (type === 'disease').
- Если болезней нет — блок «Источники» скрывается полностью.
- Визуальный стиль medication-чипов отличается от disease-чипов (акцентный цвет).

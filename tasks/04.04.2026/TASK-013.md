# TASK-013 — Генерация и импорт пенициллинов из vidal-db

> **Модуль:** `medications/data`
> **Дата начала:** 04.04.2026
> **Статус:** ✅ DONE
> **Приоритет:** HIGH

---

## 📋 Описание задачи

Найти в `vidal-db` препараты пенициллинового ряда, сгенерировать для них JSON в формате модуля препаратов и загрузить их в нашу БД через универсальный импортёр `scripts/import_medications_json_batch.py`.

### Источник отбора

Основной критерий отбора — клинико-фармакологические группы `ClPhGroups`:

- `169` — Антибиотик группы пенициллинов, разрушающийся пенициллиназой
- `170` — Антибиотик группы пенициллинов широкого спектра действия, разрушающийся пенициллиназой
- `172` — Антибиотик группы пенициллинов, резистентный к пенициллиназе
- `173` — Антибиотик группы пенициллинов широкого спектра действия, резистентный к пенициллиназе
- `175` — Антибиотик группы пенициллинов широкого спектра действия с ингибитором бета-лактамаз
- `1421` — Антибиотик - полусинтетический пенициллин в комбинации с ингибитором бета-лактамаз

Предварительно найдено `32` уникальных `DocumentID`.

---

## ✅ Что нужно сделать

1. Создать `scripts/generate_penicillin_jsons.py`
2. Сохранять JSON в `src/modules/medications/data/penicillins`
3. Использовать текущие правила:
   - очистка HTML
   - нормализация `routeOfAdmin`
   - `fullInstruction` как plain text
   - фильтрация пустых записей `pediatricDosing`
4. Выполнить импорт через `scripts/import_medications_json_batch.py`
5. Проверить количество импортированных препаратов и корректность ключевых полей

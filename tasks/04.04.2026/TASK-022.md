# TASK-022 — Исправить редактирование фильтров при пустой выдаче

> **Модуль:** `medications`  
> **Дата начала:** 04.04.2026  
> **Статус:** ✅ DONE  
> **Приоритет:** HIGH

---

## 📋 Описание задачи

При пустом результате фильтрации пользователь не мог изменить выбранные фильтры: кнопка `Фильтры` нажималась, но панель не отображалась, и оставался только полный сброс.

### Причина

Панель фильтров рендерилась только при условии `(total > 0 || isLoading)`, поэтому при `total = 0` она скрывалась, даже если `showFiltersPanel = true`.

---

## ✅ Что сделано

- В `MedicationsModule.tsx` убрано ограничение по `total` для рендера панели фильтров.
- Панель фильтров теперь открывается по кнопке `Фильтры` даже при пустой выдаче.
- Пустое состояние (`Ничего не найдено`) теперь корректно учитывает фильтр `withPediatricDosing`, чтобы UI не показывал `Справочник пуст` при активном фильтре.

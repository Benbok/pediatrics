# TASK-019 — Генерация и импорт карбапенемов из vidal-db

> **Дата начала:** 04.04.2026
> **Статус:** ✅ DONE
> **Приоритет:** HIGH

---

## 📋 Описание задачи

Найти в `vidal-db` препараты группы карбапенемов, сгенерировать для них JSON в формате модуля препаратов и загрузить их в нашу БД через универсальный импортёр `scripts/import_medications_json_batch.py`.

### Источник отбора

Используется ATC-префикс `J01DH*`:

- `J01DH02` — меропенем
- `J01DH03` — эртапенем
- `J01DH04` — дорипенем
- `J01DH05` — биапенем
- `J01DH51` — имипенем и циластатин

Предварительно найдено `17` уникальных `DocumentID`.

---

## ✅ Что нужно сделать

1. Создать `scripts/generate_carbapenem_jsons.py`
2. Сохранять JSON в `src/modules/medications/data/carbapenems`
3. Использовать текущие правила:
   - очистка HTML
   - нормализация `routeOfAdmin`
   - `fullInstruction` как plain text
   - фильтрация пустых записей `pediatricDosing`
4. Выполнить импорт через `scripts/import_medications_json_batch.py`
5. Проверить количество импортированных препаратов и корректность ключевых полей

## 📋 Описание задачи

Исправить поведение во вкладке Препараты модуля База знаний: при добавлении/удалении препарата из избранного иконка/состояние должны обновляться сразу без ручного обновления экрана.

### Контекст

Сейчас UI отображает изменение только после полной перезагрузки страницы, что указывает на отсутствие корректного обновления клиентского состояния (кэш/локальный список) после мутации.

### Ожидаемый результат

После клика по избранному в списке Препаратов состояние карточки обновляется мгновенно, а данные остаются консистентными при последующей навигации и рефетче.

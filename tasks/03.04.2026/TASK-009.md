## 📋 Описание задачи

Провести полный аудит паттернов путей введения в базе данных vidal-db (поле `ZipInfo` таблицы `Product` и `Dosage` таблицы `Document`) и привести в соответствие нашу схему `RouteOfAdmin`.

### Контекст

В vidal-db маршруты введения хранятся как свободный текст на русском языке внутри поля `ZipInfo` (например: `"р-р д/в/в и в/м введения 250 мг/1 мл"`). Нашa схема использует строго типизированный enum.

**Обнаруженная проблема:** при batch-импорте аминогликозидов:
- Генератор (`generate_aminoglycoside_jsons.py`) выдавал `iv` вместо `iv_infusion`
- Некоторые паттерны не покрывались (опталмические капли → `topical`, назальные спреи → `intranasal`)
- Для `р-р д/инъекц` (без явного в/в или в/м) — маршрут не определялся

Сейчас исправлены только аминогликозиды. Нужна **единая надёжная матрица нормализации** для всех будущих batch-импортов.

### Ожидаемый результат

1. Полная матрица маппинга ZipInfo-паттернов → `RouteOfAdmin` enum для Python-парсера
2. Обновлённая функция `infer_route()` в `generate_aminoglycoside_jsons.py`  
3. Обновлённая функция `normalize_route()` в `import_medications_json_batch.py`
4. Все 18 записей аминогликозидов в `dev.db` имеют корректный `route_of_admin`
5. TypeScript компилируется без ошибок

### ZipInfo → RouteOfAdmin маппинг

Приоритет проверки **сверху вниз** (первое совпадение побеждает):

| ZipInfo-паттерн (подстрока) | RouteOfAdmin | Примечание |
|----------------------------|--------------|------------|
| `в/в` + `капельно` | `iv_infusion` | Явно капельное в/в |
| `в/в` + `струйно` | `iv_bolus` | Явно струйное в/в |
| `в/в` + `медленно` | `iv_slow` | Медленное в/в |
| `в/в` + `в/м` | `iv_infusion` | Приоритет в/в при комбинации |
| `в/в` (одиночное) | `iv_infusion` | По умолчанию в/в → капельное |
| `р-р д/инф`, `р-р д/инфузий`, `лиофилизат д/пригот. р-ра д/инф`, `конц. д/пригот. р-ра д/инф`, `концентрат д/пригот. р-ра д/инф` | `iv_infusion` | Раствор/лиофилизат для инфузий |
| `в/м` (одиночное, без в/в) | `im` | Внутримышечно |
| `р-р д/инъекц` + `шприцы` (малый объём ≤1 мл) | `sc` | Шприц-ручки → подкожно |
| `р-р д/инъекц` (без явного в/в или в/м) + контекст Dosage содержит `в/в` | `iv_infusion` | Fallback по тексту дозирования |
| `р-р д/инъекц` (без явного в/в или в/м) + контекст Dosage содержит `в/м` | `im` | Fallback по тексту дозирования |
| `р-р д/инъекц` (без контекста) | `im` | Default для дженерик инъекций |
| `порошок д/пригот. р-ра д/инъекц`, `лиофилизат д/пригот. р-ра д/инъекц` | `im` | Default порошок → в/м |
| `п/к`, `подкожн`, `подкожного введения` | `sc` | Подкожно |
| `таб`, `капс`, `сироп`, `сусп`, `гран`, `эликсир`, `пор.` (без инъекц), `р-р оральн`, `р-р д/приема` | `oral` | Пероральные формы |
| `аэрозоль д/ингал`, `порошок д/ингал`, `р-р д/ингал`, `небулайзер`, `ингаляц` | `inhalation` | Ингаляционные |
| `глазн`, `офтальм`, `конъюнктив`, `глазные капли`, `капли глазные` | `topical` | Офтальмологические → наружный |
| `ушные`, `капли ушные`, `ушн` | `topical` | Отические → наружный |
| `назаль`, `назальн`, `интраназальн`, `спрей назал` | `intranasal` | Интраназальные |
| `ректальн`, `суппозитории рект`, `супп. рект`, `свечи рект` | `rectal` | Ректальные |
| `вагинальн`, `супп. вагин`, `вагинальные` | `topical` | Вагинальные → наружный |
| `сублингв`, `подъязычн`, `спрей подъязычный`, `под язык` | `sublingual` | Сублингвальные |
| `трансдерм`, `пластырь`, `накожный` | `transdermal` | Трансдермальные |
| `наружн`, `местн`, `крем`, `мазь`, `гель`, `лосьон`, `линимент`, `шампунь` | `topical` | Наружные |

# Тесты бэкенда прививок

Этот набор тестов проверяет корректность работы бэкенда для записей прививок.

## Что тестируется

1. **Добавление одной записи не создает смежные записи**
   - Проверяет, что при добавлении одной записи прививки не создаются дополнительные записи

2. **Добавление нескольких записей работает независимо**
   - Проверяет, что каждая запись создается независимо и не влияет на другие

3. **Редактирование записи работает корректно**
   - Проверяет, что обновление записи через `upsert` корректно изменяет данные

4. **Удаление записи работает корректно**
   - Проверяет, что удаление одной записи не затрагивает другие записи

5. **Удаление несуществующей записи вызывает ошибку**
   - Проверяет обработку ошибок при попытке удалить несуществующую запись

6. **Upsert создает новую запись если её нет**
   - Проверяет, что `upsert` создает новую запись при отсутствии существующей

7. **Upsert обновляет существующую запись**
   - Проверяет, что `upsert` обновляет существующую запись вместо создания дубликата

## Запуск тестов

```bash
npm run test:vaccination
```

Или напрямую:

```bash
node tests/vaccination-records.test.cjs
```

**Примечание**: Если возникает ошибка с `better-sqlite3`, выполните:
```bash
npm rebuild better-sqlite3
```

## Структура тестов

Тесты используют отдельную тестовую базу данных (`prisma/test.db`), которая создается автоматически при запуске и не влияет на основную базу данных разработки.

Каждый тест:
1. Очищает тестовые данные
2. Создает тестового ребенка
3. Выполняет операцию (создание/обновление/удаление)
4. Проверяет результат

## Добавленные методы

Для полноценного тестирования был добавлен метод удаления записи:

- **Backend**: `db:delete-record` в `electron/database.cjs`
- **Frontend API**: `deleteRecord(childId, vaccineId)` в `electron/preload.cjs`

## Пример использования API

```javascript
// Создание/обновление записи
await window.electronAPI.saveRecord({
  childId: 1,
  vaccineId: 'hep_b_1',
  isCompleted: true,
  completedDate: '2020-01-15',
  vaccineBrand: 'Энджерикс В',
  notes: 'Первая доза',
  dose: '1',
  series: 'ABC123',
  expiryDate: '2025-01-15',
  manufacturer: 'GSK'
});

// Удаление записи
await window.electronAPI.deleteRecord(1, 'hep_b_1');

// Получение всех записей
const records = await window.electronAPI.getRecords(1);
```
